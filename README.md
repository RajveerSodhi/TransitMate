# TransitMate
![TransitMate Rajveer's Logo](https://github.com/RajveerSodhi/TransitMate/assets/65150031/ba89af17-a3df-40de-b07c-1c2258586ca6)

A car-pooling app where the user selects a driver based on their personality.

This project was created for the final project of COSC 341 (Human-Computer Interaction) along with Mithish Ravisankar Geetha and Amey Karmakar. I continued to work on this to make minor improvements.
